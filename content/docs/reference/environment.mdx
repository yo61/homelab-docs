## Install media

- **Install disk:** `/dev/nvme0n1`.
- **Installer image:** resolved by talhelper from the extension set in
  `talconfig.yaml` (`siderolabs/amdgpu` + `iscsi-tools` + `util-linux-tools`) →
  factory `metal-installer`, schematic
  `3a33ec6dfc8cfd61d2a3db3caf97894f31e913952d71ce3c3fbbe565a3f08339`. If the
  `metal-installer` hash differs at genconfig time, the extension set is wrong —
  stop and fix, or you rebuild without `iscsi-tools`.

## Switch / LAG

Each node is a two-port UniFi LACP bond:

| Node | Ports | Leader (odd) |
| --- | --- | --- |
| `k8s-01` | 17, 18 | 17 |
| `k8s-02` | 19, 20 | 19 |
| `k8s-03` | 21, 22 | 21 |

`unifictl` toggles a whole bond by its leader port, so `--leader 17,19,21` acts on
all three. Aggregation must be off while a node is in Talos maintenance mode — see
[Talos rebuild gotchas → LACP](/explanation/talos-rebuild-gotchas#lacp).
